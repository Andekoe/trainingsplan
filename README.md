# Trainingsplan

A mobile-friendly workout tracker app for gym users. Track your progress through push and pull training sessions with an intuitive interface designed for phones.

## Features

- **Mobile-first design**: Optimized for phone usage in the gym
- **4 training days**: Push A, Pull A, Push B, Pull B
- **Progress tracking**: Visual progress rings and completion counters
- **Set logging**: Expandable cards to log weights and reps
- **Local state**: No backend required, works offline
- **Dark theme**: Easy on the eyes in gym environments
- **PWA support**: Install as app on iOS home screen with offline caching
- **Motivation boost**: Celebration screen after completing full workouts

## Setup

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. Start your workout!

## PWA Installation

This app can be installed as a Progressive Web App (PWA) on iOS devices:

1. Open the app in Safari on iOS
2. Tap the Share button (square with arrow)
3. Select "Add to Home Screen"
4. The app will appear on your home screen with a custom icon

The PWA includes offline caching, so the app works even without internet connection.

## Usage

- Tap on exercise cards to expand and log sets
- Tap again when expanded to mark as complete
- Use the tabs to switch between training days
- Reset session to start over
- Complete session to see the finish screen

## Deployment

This is a static site, perfect for GitHub Pages:

1. Upload to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Access via `https://yourusername.github.io/repository-name/`
