# App Icons

The app includes PWA icons in the following sizes:
- 180x180 (iOS home screen icon)
- 192x192 (Android/PWA standard)
- 512x512 (Android/PWA high-res)

Current icons are simple dumbbell designs created with SVG and converted to PNG.

To replace with custom icons:
1. Create PNG files in the sizes above
2. Place them in the `icons/` directory
3. Update `manifest.json` if needed

The SVG source for the current dumbbell icon is available in the project files.
